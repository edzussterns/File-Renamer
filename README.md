# File Renamer

A professional batch file renaming tool built with PyQt6.

## Features

- Numbered renaming with zero padding
- Template-based renaming with placeholders
- Find & Replace mode
- Advanced Regex mode
- Simulation (preview only)
- Undo support
- GitHub update checking

## Modes

### Numbered
Sequential renaming with configurable start number and zero padding.

### Template
Use placeholders:
- `{n}` – number
- `{title}` – title input
- `{original}` – original filename
- `{ext}` – file extension

### Find & Replace
Replace specific text in filenames.

### Advanced (Regex)
Pattern-based renaming using regular expressions.
